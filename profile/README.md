# Omicsy

**Omicsy** is a cloud-based metagenomics pipeline platform. It lets researchers upload sequencing datasets, organise them into projects, and run standardised pipeline steps — quality filtering, assembly, binning, and more — without managing any infrastructure.

Pipelines execute as containerised workflows on Google Cloud, so results are reproducible and scalable from a single sample to entire cohorts.

## What you can do with Omicsy

- **Upload sequencing data** — drag-and-drop raw reads directly from your browser; files are stored securely in the cloud
- **Organise into projects** — group datasets by study, sample type, or any structure that fits your workflow
- **Run pipeline steps** — submit quality filtering, assembly, binning, and other standardised steps with a few clicks
- **Track job progress** — real-time status updates and milestone tracking for every running job
- **Review AI-assisted digests** — automatically generated summaries of completed pipeline runs
- **Manage credits and billing** — transparent per-job cost estimates before you submit
